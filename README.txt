Loan Approval Prediction App

1. Put your training data as train.csv
2. Run: python train_model.py   → creates loan_model.pkl
3. Put your test data as test.csv
4. Run: python predict_from_test_file.py   → creates predictions.csv
5. Or run the Streamlit UI:
   streamlit run app.py

Ensure Python and pip are installed. Install dependencies:
pip install -r requirements.txt
